# how to use


## dockerコマンドについて
### 起動コマンド
```
docker compose up -d
```

### 起動状態確認
```
docker ps
```

### 終了コマンド
```
docker compose down
```

### 起動中のコンテナに入る
```
docekr compose exec [CONTAINER_NAME] bash
```
